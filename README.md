# Tic-Tac-Toe Game

A simple, interactive Tic-Tac-Toe game built with HTML, CSS, and JavaScript. Play against another player and enjoy a classic game with a modern touch.

**Features:**

- **Responsive design**
- **Player vs Player mode**
- **Dynamic game updates**
- **Intuitive UI**

**Technologies Used:**

- **HTML5**
- **CSS3**
- **JavaScript**
